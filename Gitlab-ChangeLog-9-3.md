# Edit Issue Description Inline, Without Losing Context

如果 issue 内容因为讨论的内容发生变动，而在编辑的时候又因为跳页而没法查看原来的讨论内容

当然这个问题也可通过多开一个 tab 来解决，如果不用多开那可多好

通过这一特性，直接在当前页面编辑 issue 内容，不用跳页，不用开 tab
